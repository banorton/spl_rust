# Rust Learning Resources

## Official Documentation
- [The Rust Book](https://doc.rust-lang.org/book/) - Comprehensive guide for beginners
- [Rust by Example](https://doc.rust-lang.org/rust-by-example/) - Learn through runnable examples
- [Rustlings](https://github.com/rust-lang/rustlings) - Interactive exercises
- [Standard Library](https://doc.rust-lang.org/std/) - Complete API reference

## Advanced Topics
- [Async Book](https://rust-lang.github.io/async-book/) - Asynchronous programming
- [Nomicon](https://doc.rust-lang.org/nomicon/) - Unsafe Rust and dark magic
- [Rust Reference](https://doc.rust-lang.org/reference/) - Language specification

## Practice & Community
- [Exercism](https://exercism.org/tracks/rust) - Mentored coding practice
- [Rust Playground](https://play.rust-lang.org/) - Online compiler
- [This Week in Rust](https://this-week-in-rust.org/) - Weekly newsletter
- [/r/rust](https://www.reddit.com/r/rust/) - Reddit community

## Key Crates
- [Serde](https://serde.rs/) - Serialization framework
- [Tokio](https://tokio.rs/) - Async runtime
- [Clap](https://docs.rs/clap/) - CLI argument parsing
- [Rayon](https://docs.rs/rayon/) - Data parallelism
